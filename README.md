# **_Syzygy_: An enhanced Friedman-Nemenyi framework that accounts for domain-specific metric distortions in ML/DL evaluations**

![Syzygy](syzygy-card.png)

Syzygy is a refined implementation of the Friedman-Nemenyi-Gusmao procedure, incorporating adjustments for ML/DL metric biases to produce robust multi-model rankings. Syzygy consists modern variant of the Friedman-Nemenyi test - the Friedman-Nemenyi-Gusmao Test - with bias-aware corrections tailored to machine-learning and deep-learning evaluation metrics, providing reliable statistical ranking across multiple models.

For a detailed mathematical formulation of the Friedman-Nemenyi-Gusmao method, please refer to www.gusmaolab.org.

### **Syzygy Input and Output:**
* **Input:** A raw TSV-file where columns represent distict methods or techniques (yield values to be tested) and each row contains the values output by these methods **on the same input data**.
* **Output:** A plain file containing the test statistics and method's ranking. Optionally, also more details on the test statistics and a pdf figure, generated with ``LaTeX``, representing a diagonal matrix of methods where the methods on the columns were highly ranked (with a certain significance, shown on the cells) than other methods.

## **Requirements**

Syzygy has very few requirements, most of which are usually used by any standard **python** environment:
* Pandas (v. 2.2.x or higher): https://pandas.pydata.org/
* Numpy (v. 1.17.x or higher): https://numpy.org/
* Scipy (v. 1.4.x or higher): https://www.scipy.org/
* Setuptools (python installation): https://pypi.org/project/setuptools/

The installation of all the requirements can be easily done with python package manager PyPi:

```bash
pip install -U setuptools
python -m pip install --user numpy scipy matplotlib ipython jupyter pandas sympy nose
```

If you would like to generate the pdf automatically from the ``LaTeX`` outputs, please install ``LaTeX``: https://www.latex-project.org/

---

## **Installation**

After making sure all the dependencies are installed. Simply execute the following steps:

```bash
pip install syzygy
```

If you would like to install from the source code please do:

```bash
git clone https://github.com/eggduzao/Sygyzy.git
cd Syzygy
python setup.py install --user
```

:warning: Syzygy has been tested and is supported only for python 3.X.X+. Syzygy is **not** supported on Windows systems.
